# Aria Yousefi - Personal Portfolio

This repository contains the source code for my personal portfolio website. Built with React and Material-UI, it serves as a dynamic showcase of my work as a software engineer with a passion for urban science. The centerpiece of the site is an interactive 3D cube carousel that provides a unique and engaging way to navigate through my projects, resume, and personal bio.
